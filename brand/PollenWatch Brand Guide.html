PollenWatch

The brand identity for PollenWatch — a Home Assistant integration that aggregates pollen & allergy forecasts from multiple sources into one combined severity reading.

The mark

One bloom, one reading

A multi-color bloom shedding pollen, with a severity needle that starts at the flower's heart and sweeps a green→amber→red dial. The dial reads like a warning gauge — higher is worse — so the mark never feels cheerful about a rising count.

Product system

Combined severity scale

PollenWatch resolves many sources into a single four-step index. The needle and every status surface use the same ramp, so a glance at any size tells the same story.
